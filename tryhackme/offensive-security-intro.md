# Offensive Security Intro

## First Hack

Using the CLI-application Gobuster to brute-force a website to find hidden directories and pages.

Gobuster takes a list of potential page or directory names and tries accessing a website with each of them.

1. Open the terminal
2. Run gobuster with a specified URL (-u) and wordlist (-w)

## Careers

Penetration tester - responsible for testing products, finding exploitable security vulnerabilities

Red teamer - Plays the role of adversary, attacking an organization and providing feedback from an enemy's perspective

Security engineer - design, monitor and maintain security controls, networks and systems to help prevent cyberattacks
